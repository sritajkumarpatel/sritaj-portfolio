# Portfolio Images

Place your images here for use in your portfolio.

## Required Images

### Profile Photo
- **File:** `profile.jpg` or `profile.png`
- **Recommended:** 400x400px square
- **Usage:** Hero section avatar, About section

### Project Screenshots
- **File:** `project-{name}.jpg` or `project-{name}.png`
- **Recommended:** 800x600px
- **Usage:** Project cards in Projects section

## Supported Formats
- JPG/JPEG
- PNG
- SVG (for logos)
- WebP

## Tips
- Use compressed images for faster loading
- Keep consistent aspect ratios for project screenshots
- SVG logos work best for company/certification badges
